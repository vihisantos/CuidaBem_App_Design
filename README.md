## Sobre

**CuidaBem** e um aplicativo mobile/web que conecta idosos a cuidadores de confianca. Este repositorio contem o bundle de codigo extraido do design no Figma, com componentes estilizados usando React, Radix UI e Tailwind CSS.

---

## Tech Stack

| Tecnologia | Proposito |
|---|---|
| **React 18** | Biblioteca de componentes |
| **Radix UI** | Primitivos acessiveis |
| **Tailwind CSS 4** | Estilizacao utilitaria |
| **Vite 6** | Build tool |
| **Figma** | Design original |

---

## Funcionalidades

- Design system completo extraido do Figma
- 10 paginas: Login, Registro, Home, Busca, Perfil, Pedidos, Perfil do Usuario, Confirmacao de Pedido, Home do Cuidador, Mapa
- Componentes React + Radix UI
- Tema personalizavel
- Layout responsivo (mobile + desktop)
